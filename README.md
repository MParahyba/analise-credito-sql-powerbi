# Sistema de Análise de Risco de Crédito e Carteira Bancária 📊🚀

Este é o meu primeiro projeto prático integrando banco de dados relacional e inteligência de negócios, simulando um cenário real do setor financeiro.

## 🛠️ Tecnologias Utilizadas
* **SQL Server:** Modelagem de dados, chaves primárias/estrangeiras e lógica de negócio.
* **Power BI:** Extração de dados (Power Query), modelagem e desenvolvimento de dashboard interativo.

## 📈 O que foi desenvolvido

### 1. Banco de Dados (SQL Server)
* Modelagem das tabelas de Clientes, Contas e Empréstimos.
* Criação de uma `VIEW` utilizando `INNER JOIN` para unificar os dados do banco.
* Implementação de uma regra de negócio com `CASE WHEN` para classificar automaticamente os clientes em faixas de Score: **Baixo Risco**, **Médio Risco** e **Alto Risco**.

### 2. Dashboard (Power BI)
* Criação de cartões para exibição de valores macros (`Saldo em Conta` e `Total Emprestado`).
* Gráfico de barras indicando o volume financeiro por classificação de risco.
* Gráfico de rosca demonstrando que 89% do capital em conta está concentrado no segmento PJ (Pessoa Jurídica).
* Filtros dinâmicos por segmentação de região geográfica.
